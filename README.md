1. Nombre Oficial del Proyecto
TECHTRACK: SISTEMA CENTRALIZADO DE GESTIÓN DE ACTIVOS DE TI (ITAM)

2. Explicación General del Proyecto (Enfoque Core Business y MVP)

¿Qué es? Una plataforma web orientada a departamentos de TI para administrar el ciclo de vida completo de la infraestructura tecnológica de una organización, abarcando tanto hardware físico como activos digitales (software).
Componentes Principales (El Verdadero Core):
Gestor de Inventario Universal: Un módulo CRUD robusto que unifica en una sola base de datos los equipos físicos (laptops, servidores, periféricos) y las licencias de software institucionales.
Motor de Asignaciones: La lógica central que vincula a un usuario (empleado) con un activo, registrando fechas de entrega, retornos esperados y un historial inmutable de responsables.
Control de Ciclo de Vida y Alertas: Un sistema de monitoreo que rastrea el estado del equipo (Disponible, Asignado, En Mantenimiento, Obsoleto) y genera notificaciones automáticas visuales en el Dashboard basadas en umbrales de stock mínimo o fechas de expiración de licencias.
Herramientas Operativas (Trazabilidad QR Simplificada): Funcionalidades de soporte para agilizar el trabajo de campo. El sistema genera automáticamente una imagen de código QR por cada activo físico. Al escanear este QR con la cámara nativa de cualquier smartphone, el técnico es redirigido a la URL del perfil del equipo en el sistema web, permitiendo auditorías rápidas sin la complejidad de programar un escáner interno en la aplicación.
